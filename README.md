## 📦 节点一览 · 速览（2025-12-18 更新）
| 节点 | 功能 | 亮点 |
|---|---|---|
| **图像稳定水印** 💕 | 单张/批量加透明水印 | 下拉选图+一键刷新 |
| **视频稳定水印** 🌸 | 4K@50 fps 流式水印 | 内存&lt;3 GB、音轨无损 |
| **视频强制落盘+尺寸** ✨ | WAN/任何对象→真实路径 | 零属性依赖 |
| **一键保存 JPG** 🖼 | 保存+同步提示词 txt | 支持日期变量 |
| **简易提示词&分辨率** 🎀 | 比例一键锁、空潜输出 | 8 倍数对齐 |
